# Bicep Planning Files

This folder contains implementation plans generated by the `bicep-plan` agent.

## Purpose

Implementation plans are machine-readable markdown files that specify:

- Azure resources to deploy
- Azure Verified Module (AVM) versions
- Dependencies between resources
- Configuration parameters
- Cost estimates

## Usage

1. Use `azure-principal-architect` agent for architecture assessment
2. Use `bicep-plan` agent to generate implementation plan
3. Plans are saved here as `INFRA.{goal}.md`
4. Use `bicep-implement` agent to generate Bicep code from the plan

## File Naming

- `INFRA.ecommerce-platform.md` - E-commerce platform plan
- `INFRA.web-app.md` - Web application plan
- `INFRA.data-platform.md` - Data platform plan
