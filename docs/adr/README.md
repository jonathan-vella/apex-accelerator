# Architecture Decision Records

This folder contains Architecture Decision Records (ADRs) documenting important technical decisions.

## What is an ADR?

An ADR is a document that captures an important architectural decision made along with its context and consequences.

## Creating New ADRs

Use the `adr-generator` agent in Copilot:

1. Press `Ctrl+Shift+A` in VS Code
2. Select `adr-generator`
3. Describe the decision you want to document

## ADR Template

```markdown
---
title: "ADR-NNNN: [Decision Title]"
status: "Proposed"
date: "YYYY-MM-DD"
authors: "[Stakeholder Names/Roles]"
---

## Status

Proposed

## Context

[Problem statement and requirements]

## Decision

[Chosen solution with rationale]

## Consequences

### Positive

- [Benefits]

### Negative

- [Trade-offs]

## Alternatives Considered

- [Other options and why rejected]
```
